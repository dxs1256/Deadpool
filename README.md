**功能：**

*   从远程 URL 获取 SOCKS5 代理。
*   多线程检查代理可用性。
*   去重、保存有效代理。
*   可选：定时更新。

**用法：**

*  配置 config.toml。

*  运行 python main.py。

**输出：**

*  控制台日志。

*  有效代理保存至 lastData.txt。
